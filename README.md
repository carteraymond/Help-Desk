# Help Desk

This Help Desk application provides a centralized platform for managing support tickets, tracking issues, and coordinating resolutions between support staff and users. It streamlines the process of reporting problems, assigning tasks, and monitoring progress to ensure efficient customer service delivery.


## Instructions for Build and Use

[Software Demo](Put_Your_Video_Link_Here)

Steps to build and/or run the software:

1. Download helpdesk
2. Run "npm install" in terminal
3. navigate to helpdesk-server "cd helpdesk-server"
4. run "node server.js"

Instructions for using the software:

1. Fill out ticket title and description
2. click submit ticket
